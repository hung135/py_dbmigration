How to Deploy:
    1: make buildCentos6

Upload to PyPi:
    1. commit 
    2. git tag <next tag number> # git tag --to list all tags
    3. make build
    4. make release

Github-Manager:
    Uploading:
        You need a github token which can be generated from github.com/settings/tokens and generate one
        To upload:
            python github-manager.py -e https://github.cfpb.gov -k <token> -r data/py_dbmigration -c "Data_load executable" -m <message_for_release> -a <path-to-asset>
        To update a release:
            python github-manager.py -e -k <token> -r <repo> -u -t <tag_id> -a <path-to-asset>
    Downloading:
            python github-manager.py -e -r <repo> -dl <tag> -o <output_dir>
